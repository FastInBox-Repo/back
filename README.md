# FastInBox Back

Backend inicial da plataforma FastInBox, configurado com NestJS + TypeScript.

## Scripts

```bash
npm run start:dev
npm run build
npm run lint
```

## Ambiente local

- API: `http://localhost:4001`
- Frontend esperado: `http://localhost:3001`
- Postgres Docker: porta `5451`
- Redis Docker: porta `6380`

## Infra local

```bash
docker compose up -d
```

## Observacao

Este repositorio esta apenas inicializado e configurado. O desenvolvimento funcional ainda nao comecou.
